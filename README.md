# Smart Gym
Es una aplicación desarrollada por seis alumnas del Instituto de Formación Técnica Superior N.° 24 (IFTS N.° 24) como parte de un proyecto académico.

La aplicación tiene como objetivo brindar una solución tecnológica para la gestión y seguimiento de actividades relacionadas con un gimnasio, ofreciendo una experiencia intuitiva y moderna para sus usuarios.

## Tecnologías utilizadas

Este proyecto fue desarrollado utilizando las siguientes tecnologías:

-Node.js
-Angular
-Ionic Framework
-Capacitor
-HTML5
-CSS3
-TypeScript
